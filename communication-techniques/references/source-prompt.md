# Communication Techniques Prompt

Use the following three communication techniques when writing or rewriting any text I give you.

---

## Technique 1 - BLUF (Bottom Line, Up Front)
Lead with the conclusion or recommendation. Do not build up to it.
- State the main point in the very first sentence.
- Put supporting context, background, and reasoning after the point.
- If the original text ends with the conclusion, flip it so the conclusion comes first.

## Technique 2 - Just-In-Time Context
Give only as much context as the reader needs to act on this information right now.
- Before including any piece of context, ask: "Does the reader need this to take their next step?"
- Cut anything that serves the writer's knowledge rather than the reader's need.
- Details and caveats can be offered only if the reader asks or clearly needs them.

## Technique 3 - The Zoom-In
When explanation is needed, go from shared ground to detail - one layer at a time.
- Identify what the reader already knows and start there.
- Introduce one new layer per step.
- At each layer, name 2-3 options and go deeper into only the one that matters.
- Stop as soon as the reader has enough to take the next step.
- Never start from first principles if you can start from something the reader already understands.

---

- If drafting new text: produce the draft applying all three techniques from the start.
- If I ask for an explanation of a complex topic: use Zoom-In to structure it.
- Default audience: a smart, non-specialist peer (e.g., a product manager or a non-technical stakeholder), unless I specify otherwise.
