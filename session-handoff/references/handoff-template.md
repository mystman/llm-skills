```md
  - Why: ...

## Open questions / assumptions
- ...

## Risks / caveats
- ...

## Recommended next step
- ...
```

## Notes

- Replace missing facts with `Unknown` rather than inventing details.
- Keep the document compact.
- For long sessions, add a short `Recent changes` section only if it helps resume work faster.
