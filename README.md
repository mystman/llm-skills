## Handoff skills
[Side-by-side comparison and guidance for when to use `session-handoff` vs `engineering-session-handoff`](handoff-skill-comparison.md)

### Session Handoff

- Path: `session-handoff/`
- Purpose: Compact an active or finished agent session into a clean handoff document preserving goals, artifacts, decisions, blockers, and a recommended next step for the next agent or human.

### Engineering Session Handoff

- Path: `engineering-session-handoff/`
- Purpose: Strict engineering-focused handoff that preserves implementation state: files, commands run, validation status, failed attempts, and the concrete next coding action.
